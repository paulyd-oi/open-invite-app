ROOT CAUSE
----------
React Native's fetch implementation silently drops the uppercase 'Cookie' header.
Better Auth expo client uses lowercase 'cookie' header + credentials:'omit' + expo-origin header.
Our $fetch wrapper was using uppercase 'Cookie' and credentials:'include'.
