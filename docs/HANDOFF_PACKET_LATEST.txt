SUMMARY
-------
Removed "swings down then swings up" animation artifact on Welcome
screen load by replacing spring-based animation with duration-based.

ROOT CAUSE
----------
FadeInUp.springify().damping(20).stiffness(90) still allowed spring
overshoot on translateY, causing visible swing/rebound on load.

FIX
---
Replaced springify() chain with duration-based entrance animation:

FROM:
  FadeInUp.delay(delayMs).springify().damping(20).stiffness(90)

TO:
  FadeInUp.delay(delayMs).duration(280)

Duration-based animation cannot overshoot - it interpolates linearly
from start to end position over 280ms.
