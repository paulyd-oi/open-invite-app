# Open Invite - TestFlight Beta Testing Guide

## What to Test

Welcome to Open Invite! This is a social calendar app where you can share your plans and discover what your friends are up to. Here's what we'd love your feedback on:

### Core Features to Test

**1. Account & Authentication**
- Sign up with email and verify your account
- Sign in/out functionality
- Password reset flow

**2. Event Creation**
- Create one-time, weekly, and monthly recurring events
- Add location, description, and custom emoji
- Set event visibility (all friends or specific groups)
- Test different event categories

**3. Calendar Views**
- Switch between Compact, Stacked, Details, and List views
- Pinch-to-expand calendar rows
- Scroll through months (12 back, 24 forward)
- Tap dates to see "Who's Free?"

**4. Friends & Social**
- Add friends by email or phone number
- Accept/decline friend requests
- Create and manage friend groups
- View friend profiles and their events

**5. Event Interactions**
- Request to join events
- Show interest with "Maybe" reaction
- Add comments (with optional photos)
- Share event memories after events end

**6. Notifications**
- Event reminders (test different reminder times)
- Friend request notifications
- Event join request notifications

**7. Calendar Sync**
- Export events to Apple Calendar
- Import events from device calendars
- Check for event conflicts

### Known Issues
- Push notifications may not work in simulator
- Some features require premium (free during beta!)

### How to Report Issues
Please include:
1. What you were trying to do
2. What happened instead
3. Device model and iOS version
4. Screenshots if possible

### Feedback We're Looking For
- Is the app intuitive and easy to use?
- Are there any crashes or bugs?
- What features are missing?
- How's the overall performance?
- Any UI/UX improvements?

Thank you for helping us make Open Invite better!
